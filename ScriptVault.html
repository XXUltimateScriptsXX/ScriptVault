📦 Installation Guide

Get Set Up in 60 Seconds

Follow the steps below to load ScriptVault into Chrome manually.

01

Open Chrome Extensions

In your Chrome browser, navigate to the extensions manager.
chrome://extensions

02

Enable Developer Mode

Toggle on Developer Mode in the top-right corner of the extensions page. This unlocks manual loading.

03

Click "Load Unpacked"

A new button will appear in the top-left. Click Load unpacked to open a folder picker dialog.

04

Select the ScriptVault Folder

Navigate to your Downloads and select the ScriptVault folder that was just downloaded.

05

You're Live!

ScriptVault should now appear in your Chrome extensions list and be accessible from the toolbar. Pin it for easy access.

✅

All done — ScriptVault is ready.

If anything isn't working, make sure Developer Mode is enabled and the correct folder was selected.
